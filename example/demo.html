🔧 集成方法

在你的网站中添加以下代码即可开始统计：

<!-- 在 </head> 标签前添加 --> <script> window.VISIT_TRACKER_API = 'http://your-domain.com/api/track'; window.VISIT_TRACKER_KEY = 'your-api-key-here'; </script> <script src="http://your-domain.com/sdk/tracker.js"></script>
注意： 请将 your-api-key-here 替换为你在管理后台获取的真实 API Key。

✨ 功能特性

  • 轻量级 - 压缩后仅几KB，不影响页面加载速度
  • 实时统计 - 实时收集和展示访问数据
  • 隐私友好 - 不使用第三方跟踪器，数据完全自主控制
  • 多维度分析 - 支持页面、来源、地理位置等多维度统计
  • 易于集成 - 只需一行代码即可完成集成

🎯 自定义事件

除了自动统计页面访问，你还可以手动跟踪自定义事件：

// 跟踪按钮点击 VisitTracker.track('button_click', { button_name: 'download', page: 'homepage' }); // 跟踪表单提交 VisitTracker.track('form_submit', { form_type: 'contact', success: true });
